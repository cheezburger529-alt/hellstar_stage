Prompt: Live Stage Website (No Signup, Unique @Username, Raise Hand + Chat)

Build a simple live stage web platform where people can join a live speaking stage with camera, microphone, chat, and raise-hand requests.

The site must be very simple, dark themed, and work on mobile and desktop browsers.

Users do NOT need to create accounts, but they must choose a unique @username before entering a stage.

Tech Stack

Frontend

React

TailwindCSS

Backend

Node.js

Express

Socket.io

Video / Audio

WebRTC

Storage

In-memory storage (no database required)

User Entry System (No Signup)

When a user opens the website they see a Join Screen.

Inputs:

Choose a @username

Rules:

must start with @

3–20 characters

letters, numbers, underscore only

must be unique in the stage

Examples:

@alex
@rockstar
@tech_guy

If username is already taken in that stage → show error:

Username already in use

The username is temporary and only lasts during the session.

Store it in localStorage so users keep it if they refresh.

Admin Login

Create a simple admin login page.

Admin password:

GG_/(())_ROCKSTAR

Speakers

Raise Hand System

Audience users can press:

Raise Hand

This sends a speaker request to the admin.

Admin sees a Requests Queue:

Example:

@alex wants to speak
[Approve] [Reject]

If approved:

user moves to speaker stage

camera/mic controls appear

User sees notification:

You were invited to speak
Speaker Mode

Speakers can:

Toggle Microphone
Toggle Camera
Leave Stage

Speakers appear in a video grid.

Example:

[ @alex ]  [ @luna ]
[ @dev ]   [ @mark ]

Use WebRTC for live video/audio.
